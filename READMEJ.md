# VSDotnet

VSDotnetはMechanical Simulation様が提供するCarSimをC#で利用するために発足させたプロジェクトです。

主な目的はUnreal Engineで提供されているように、CarSimでシミュレーションされた結果をUnity上でも表現することです。

日本代理店Virtual Mecanical Simulation様経由でMechanical Simulation様にUnreal EngineでのPluginを元に開発することの許可を得て開発を開始しました。
ただしこのプロジェクト自体に両サポートはございませんので注意ください。

Unity上での基本的な動作は完了し、デバッグを進めながらVsConnectなどのほうの開発を現在進めております。



# 対応環境
Windows x64

Visual Studio 2015以上

Unity 2019.1以上
